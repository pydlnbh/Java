# 位运算总结

| 符号 | 描述 | 运算规则                                    |
| ---- | ---- | ------------------------------------------- |
| &    | 与   | 全是1才为1                                  |
| \|   | 或   | 全是0才为0                                  |
| ^    | 异或 | 相同为0, 不同为1                            |
| ~    | 取反 | 按位取反                                    |
| <<   | 左移 | 往左进一位, 高位丢弃 (相当于乘以2)          |
| >>   | 右移 | 往右退一位, 高位补0或者符号位 (相当于除以2) |
